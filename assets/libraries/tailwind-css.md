# Radix UI

## 基本信息

- **名称**：Radix UI
- **包名**：多个包（按需安装）
- **文档**：https://www.radix-ui.com/primitives

## 说明

项目已集成 Radix UI，提供无样式的可访问性组件基础，通过 Tailwind CSS 定制样式。

## 常用包

- dialog
- dropdown-menu
- select
- popover
- tooltip
- tabs
- accordion
- slot

## 注意

shadcn/ui 基于 Radix UI 构建，优先使用 shadcn/ui 组件。
