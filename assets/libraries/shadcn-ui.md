# shadcn/ui

- **名称**：shadcn/ui
- **包名**：通过 CLI 添加组件（非 npm 包）
- **版本**：v3.6.2
- **文档**：https://ui.shadcn.com/llms.txt

## 使用方式

```bash
npx shadcn@latest add [component-name]
```

组件源码会复制到 `src/components/ui/`，可自由修改。

## 依赖

- Tailwind CSS v4（必需，已配置好构建）
- lucide-react（图标库）
