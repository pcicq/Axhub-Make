# 健身 App 首页

## 1. 简要概述

健身 App 首页是一个模拟真实移动端健身应用的示例页面,展示了现代移动应用的设计风格和交互模式。该页面采用沉浸式暗黑主题设计,配合高饱和度霓虹色强调,符合运动类应用的视觉潮流。

**核心能力:**
- 展示用户运动数据统计(卡路里、运动时长、连续天数)
- 动态环形进度条展示每日目标完成度
- 推荐课程横向滚动列表
- 底部导航栏模拟原生 TabBar 交互
- 完全可配置的主题色和用户信息

## 2. 业务需求

### 业务背景
随着健康意识的提升,健身类移动应用成为用户日常生活的重要工具。该页面作为示例,展示了如何在 Axhub 平台中实现符合现代移动端设计规范的健身应用首页,帮助产品经理和设计师快速构建原型。

### 用户需求
- 用户需要一目了然地查看今日运动数据统计
- 用户需要清晰地了解每日目标完成进度
- 用户可以浏览和选择推荐的训练课程
- 用户可以快速启动训练计划
- 用户可以通过底部导航切换不同功能模块
- 用户希望界面美观、符合运动应用的视觉风格

### 业务流程
1. 页面加载时展示用户个性化问候和头像
2. 显示今日运动数据统计(卡路里、时长、连续天数)
3. 展示每日目标完成进度(环形进度条)
4. 横向滚动展示推荐课程列表
5. 用户可点击课程卡片查看详情
6. 用户可点击"开始训练"按钮启动训练
7. 用户可通过底部导航切换不同 Tab

### 业务规则
- 每日目标默认为 500 kcal,可通过配置项自定义
- 进度条实时反映目标完成百分比
- 课程列表支持横向滚动,节省垂直空间
- 底部导航支持 4 个 Tab 切换(首页、计划、统计、我的)
- 所有交互都会触发相应的事件通知

## 3. 设计说明

### 组成结构
页面由以下部分组成:
1. **页面头部**: 用户问候语、用户名、头像
2. **统计数据卡片区**: 三个数据卡片(卡路里消耗、运动分钟、连续天数)
3. **今日计划区**: 环形进度条、目标信息、开始训练按钮
4. **推荐课程区**: 横向滚动的课程卡片列表
5. **浮动按钮**: 右下角的添加按钮
6. **底部导航栏**: 4 个 Tab 切换按钮

### 内容类型
- 用户名(可配置)
- 统计数据(卡路里、时长、天数 - 静态展示)
- 每日目标值(可配置)
- 进度百分比(可通过动作更新)
- 课程列表(来自数据源,包含标题、时长、难度、分类、封面图)
- 底部导航标签(首页、计划、统计、我的)

### 布局和样式
- **主题**: 暗色背景(#0a0a0a),配合霓虹色强调(默认 #a6ff00)
- **布局**: 垂直滚动,各区域从上到下排列
- **卡片设计**: 使用半透明背景和模糊效果(玻璃态)
- **进度条**: SVG 实现的环形进度条,支持动态更新
- **课程列表**: 横向滚动,卡片带图片背景和渐变遮罩
- **底部导航**: 固定在底部,使用 SVG 图标
- **响应式**: 适配移动端屏幕尺寸

### 交互行为
1. **点击课程卡片**:
   - 触发 `onCourseClick` 事件,传递课程对象
   - 可用于跳转到课程详情页

2. **点击开始训练按钮**:
   - 触发 `onStartWorkout` 事件
   - 可用于启动训练流程

3. **点击底部导航 Tab**:
   - 切换当前选中的 Tab
   - 触发 `onTabChange` 事件,传递 Tab 索引
   - 选中 Tab 会高亮显示(使用强调色)

4. **动作调用**:
   - `refreshData`: 刷新首页数据
   - `updateProgress`: 动态更新今日目标进度

### 状态变化
- `currentTab`: 当前选中的 Tab 索引(0-3)
- `todayProgress`: 今日目标完成进度(0-100)

## 4. Axhub API 说明

### 事件列表
| 事件名 | 描述 | 触发时机 | 参数 |
|--------|------|----------|------|
| `onCourseClick` | 点击课程卡片时触发 | 点击任意课程卡片时 | `{ course: object }` - 被点击的课程对象 |
| `onStartWorkout` | 点击开始训练时触发 | 点击今日计划中的开始按钮时 | `{}` - 空对象 |
| `onTabChange` | 切换底部标签栏时触发 | 点击底部导航任意 Tab 时 | `{ index: number }` - Tab 索引(0-3) |

### 动作列表
| 动作名 | 描述 | 参数 |
|--------|------|------|
| `refreshData` | 刷新首页数据 | 无 |
| `updateProgress` | 更新今日目标进度 | `{ progress: number }` - 进度值(0-100) |

### 变量列表
| 变量名 | 类型 | 描述 |
|--------|------|------|
| `currentTab` | `number` | 当前选中的标签页索引(0-3) |
| `todayProgress` | `number` | 今日目标完成进度(0-100) |

### 配置项列表
| 配置项ID         | 类型            | 显示名称       | 说明                        | 默认值         |
| ------------- | ------------- | ---------- | ------------------------- | ----------- |
| `userName`    | `input`       | 用户名        | 显示在页面顶部的用户名               | `'Alex'`    |
| `accentColor` | `colorPicker` | 强调色        | App 的主要强调色,影响按钮、进度条、选中状态等 | `'#a6ff00'` |
| `dailyGoal`   | `inputNumber` | 每日目标(kcal) | 每日卡路里消耗目标值                | `500`       |

### 数据项列表
| 数据项名称 | 描述 | 数据结构 |
|-----------|------|----------|
| `courses` | 推荐课程列表 | `Array<{ id: number, title: string, duration: number, level: string, category: string, image: string }>` |

**courses 字段说明:**
- `id` (number): 课程唯一标识
- `title` (string): 课程标题
- `duration` (number): 课程时长(分钟)
- `level` (string): 难度等级(如 K1、K2、K3)
- `category` (string): 分类标签(如"减脂"、"塑形"、"恢复")
- `image` (string): 封面图片 URL
