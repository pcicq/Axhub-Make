# Landing Page 技术规格文档

## 概述
使用 React 和 Tailwind CSS 构建的现代响应式着陆页，采用 shadcn/ui 设计模式。该页面展示了一个 AI 驱动的开发平台，包含多个区域：hero、功能、价格、用户评价和 FAQ。

## 组件结构

### 主布局
- **index.tsx**：主着陆页组件，协调所有区域
- 使用 AnimatedSection 包装器实现平滑滚动动画

### 区域

#### 1. Hero 区域
- 全宽 hero 配合 SVG 网格背景
- 渐变叠加层增加视觉深度
- 带有 logo 和菜单的头部导航
- 主标题和 CTA 按钮
- 响应式设计（移动端、平板、桌面）

#### 2. Dashboard 预览
- 带阴影效果的浮动预览卡片
- 绝对定位在 hero 区域下方
- 响应式尺寸

#### 3. 社会证明
- 公司 logo 网格（8 个 logo）
- 移动端 2 列，桌面端 4 列
- 灰度效果配合透明度

#### 4. Bento 区域（功能）
- 3 列网格布局（响应式）
- 6 个功能卡片：
  - AI 代码审查
  - 实时预览
  - 一键集成
  - MCP 连接
  - 并行智能体
  - 轻松部署
- 卡片采用玻璃态效果
- 背景模糊装饰

#### 5. 大型用户评价
- 单个大型评价卡片
- 头像、姓名和公司
- 居中布局

#### 6. 价格区域
- 3 个价格层级：免费版、专业版、旗舰版
- 月付/年付切换
- 带勾选标记的功能列表
- 突出显示"热门"计划
- 每个层级的 CTA 按钮

#### 7. 用户评价网格
- 3 列布局
- 共 7 条评价
- 混合卡片尺寸（大和小）
- 不同背景样式（青色、浅色、深色）

#### 8. FAQ 区域
- 手风琴式 FAQ 项目
- 6 个可展开答案的问题
- 平滑展开/折叠动画
- 背景模糊装饰

#### 9. CTA 区域
- 最终行动号召
- SVG 渐变背景
- 大标题和按钮

#### 10. Footer
- 公司信息和社交链接
- 3 列导航（产品、公司、资源）
- 响应式布局

## 样式方法

### Tailwind CSS
- 使用 CSS 变量进行主题化
- 响应式断点（md、lg）
- 使用 HSL 值的自定义调色板

### 设计令牌
- `--background`：主背景色
- `--foreground`：主文本色
- `--primary`：品牌主色
- `--secondary`：次要强调色
- `--muted`：柔和文本/背景
- `--accent`：强调高亮
- `--border`：边框颜色
- `--card`：卡片背景

### 效果
- 使用 backdrop-filter 的玻璃态效果
- 渐变叠加
- 盒阴影
- 模糊效果增加深度
- 平滑过渡

## 响应式设计

### 断点
- 移动端：< 768px
- 平板：768px - 1024px
- 桌面：> 1024px

### 移动端优化
- 堆叠布局
- 减少间距
- 较小的排版
- 简化导航
- 触摸友好按钮

## 可访问性
- 语义化 HTML 元素
- 图标的 ARIA 标签
- 键盘导航支持
- 焦点状态
- 颜色对比度合规

## 性能
- 动画无外部依赖
- 优化的 SVG 图形
- 支持懒加载
- 最小化 JavaScript
- 基于 CSS 的动画
