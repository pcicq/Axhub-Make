# Ant Design 组件展示页面
## 1. 简要概述

Ant Design 组件展示页面是一个综合性的演示页面，用于展示如何在 Axhub 页面中集成和使用 Ant Design 组件库。该页面包含了常用的 UI 组件（按钮、输入框、表格等），演示了完整的页面开发规范和最佳实践。

**核心能力：**
- 集成 Ant Design 组件库
- 展示多种常用 UI 组件的使用方法
- 提供完整的数据交互和事件处理
- 遵循 Axhub 页面开发规范

## 2. 业务需求

### 业务背景
Ant Design 是企业级 UI 设计语言和 React 组件库，广泛应用于中后台产品。该页面作为示例，展示了如何在 Axhub 平台中正确使用 Ant Design，帮助开发者快速上手。

### 用户需求
- 用户需要查看 Ant Design 各类组件的实际效果
- 用户可以与各种 UI 组件进行交互
- 用户可以查看表格数据并进行选择操作
- 用户可以触发刷新等数据操作
- 用户可以通过配置自定义页面标题

### 业务流程
1. 页面加载时展示各类 Ant Design 组件
2. 用户可以点击按钮触发事件
3. 用户可以在输入框中输入内容
4. 用户可以点击表格行进行选择
5. 用户可以点击刷新按钮重新加载数据

### 业务规则
- 表格支持分页，默认每页显示 5 条数据
- 点击表格行会高亮显示选中状态
- 刷新操作会显示加载动画（1秒）
- 所有交互都会触发相应的事件通知
## 3. 设计说明

### 组成结构
页面由以下部分组成：
1. **页面头部**：显示页面标题和说明
2. **按钮组件卡片**：展示各种类型的按钮
3. **输入组件卡片**：展示输入框、搜索框、日期选择器、开关等
4. **数据表格卡片**：展示数据表格和刷新功能

### 内容类型
- 页面标题（可配置）
- 各类按钮（主要、默认、虚线、文本、链接、危险、加载中、禁用）
- 输入控件（文本输入、搜索、日期选择、开关）
- 数据表格（包含 ID、姓名、年龄、地址、标签等字段）
- 选中行信息展示

### 布局和样式
- 使用 Ant Design 的 `Space` 和 `Card` 组件进行布局
- 垂直排列，各卡片之间有适当间距
- 表格支持分页和行选择
- 选中行会有背景色高亮
- 使用 Ant Design 默认主题样式

### 交互行为
1. **按钮点击**：
   - 点击主要按钮触发 `onButtonClick` 事件
   - 传递时间戳信息

2. **表格行点击**：
   - 点击表格行进行选择
   - 触发 `onTableRowClick` 事件，传递行数据
   - 在表格下方显示选中行的详细信息
   - 选中行会有视觉高亮

3. **刷新操作**：
   - 点击刷新按钮
   - 显示加载动画（1秒）
   - 可通过 `refreshData` 动作触发

4. **输入操作**：
   - 支持文本输入、搜索、日期选择等
   - 可与外部系统集成

### 状态变化
- `selectedRow`：当前选中的表格行数据
- `loading`：表格加载状态
## 4. Axure API 说明

### 事件列表
| 事件名 | 描述 | 触发时机 | 参数 |
|--------|------|----------|------|
| `onButtonClick` | 按钮点击时触发 | 点击主要按钮时 | `{ timestamp: number }` - 点击时的时间戳 |
| `onTableRowClick` | 表格行点击时触发 | 点击表格任意行时 | `{ row: object }` - 被点击的行数据 |

### 动作列表
| 动作名 | 描述 | 参数 |
|--------|------|------|
| `refreshData` | 刷新数据 | 无 |

### 变量列表
| 变量名 | 类型 | 描述 |
|--------|------|------|
| `selectedRow` | `object \| null` | 当前选中的行数据，未选中时为 null |

### 配置项列表
| 配置项ID | 类型 | 显示名称 | 说明 | 默认值 |
|----------|------|----------|------|--------|
| `title` | `input` | 页面标题 | 显示在页面顶部的标题 | `'Ant Design 组件展示'` |

### 数据项列表
| 数据项名称 | 描述 | 数据结构 |
|-----------|------|----------|
| `tableData` | 表格数据 | `Array<{ id: number, name: string, age: number, address: string, tags: string[] }>` |

**tableData 字段说明：**
- `id` (number)：用户 ID，唯一标识
- `name` (string)：姓名
- `age` (number)：年龄
- `address` (string)：地址
- `tags` (string[])：标签数组，用于分类

**数据示例：**
```json
{
  "tableData": [
    { "id": 1, "name": "张三", "age": 32, "address": "北京市朝阳区", "tags": ["开发", "设计"] },
    { "id": 2, "name": "李四", "age": 28, "address": "上海市浦东新区", "tags": ["产品"] },
    { "id": 3, "name": "王五", "age": 35, "address": "广州市天河区", "tags": ["运营", "市场"] }
  ]
}
```
