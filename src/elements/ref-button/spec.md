# 按钮组件
## 1. 简要概述

按钮组件是一个基础交互元素，用于展示 Axhub 组件开发规范和最佳实践。该组件提供了点击交互、状态管理、事件触发等核心功能，适用于需要用户操作触发的场景。

**核心能力：**
- 支持点击交互和计数功能
- 可配置的外观和文本内容
- 支持数据绑定和动态更新
- 完整的事件和动作系统

## 2. 业务需求

### 业务背景
作为 Axhub 平台的示例组件，需要展示完整的组件开发流程和规范要求，帮助开发者理解如何正确开发 Axhub 组件。

### 用户需求
- 用户可以通过点击按钮触发交互
- 用户可以查看和重置计数器
- 用户可以自定义按钮文本、颜色等外观
- 用户可以通过配置项和数据源控制组件行为

### 业务规则
- 计数器初始值可配置，默认为 0
- 每次点击主按钮，计数器加 1
- 重置按钮可将计数器恢复到初始值
- 所有交互都会触发相应的事件通知
## 3. 设计说明

### 组成结构
组件由以下部分组成：
1. **标题区域**：显示组件标题，颜色可配置
2. **控制区域**：包含主按钮和重置按钮
3. **消息区域**：显示当前消息内容
4. **数据展示区域**：展示绑定的数据列表（可选）

### 内容类型
- 标题文本（可配置）
- 按钮文本（可配置）
- 计数值（动态）
- 消息内容（可配置/可通过动作更新）
- 数据列表（来自数据源）

### 布局和样式
- 采用垂直布局，各区域从上到下排列
- 使用 BEM 命名规范的 CSS 类名（`axhub-button-*`）
- 主色调可通过配置项自定义
- 按钮采用现代化的圆角设计
- 支持响应式布局

### 交互行为
1. **主按钮点击**：
   - 计数器加 1
   - 触发 `onClick` 事件，传递当前计数值
   - 触发 `onCountChange` 事件，传递新计数值和动作类型

2. **重置按钮点击**：
   - 计数器重置为初始值
   - 触发 `onCountChange` 事件，传递重置后的值

3. **动作调用**：
   - 支持通过 `fireAction` 方法调用组件动作
   - 支持外部控制计数器和消息内容

### 状态变化
- `count`：计数器值，随点击和重置操作变化
- `message`：消息内容，可通过配置或动作更新
## 4. Axhub API 说明

### 事件列表
| 事件名 | 描述 | 触发时机 | 参数 |
|--------|------|----------|------|
| `onClick` | 点击按钮时触发 | 点击主按钮时 | `{ count: number }` - 当前计数值 |
| `onCountChange` | 计数器改变时触发 | 计数器值发生变化时 | `{ count: number, action: string }` - 新计数值和动作类型 |

### 动作列表
| 动作名 | 描述 | 参数 |
|--------|------|------|
| `increment` | 计数器加1 | 无 |
| `reset` | 重置计数器 | 无 |
| `setMessage` | 设置消息内容 | `{ message: string }` - 新的消息内容 |

### 变量列表
| 变量名 | 类型 | 描述 |
|--------|------|------|
| `count` | `number` | 当前计数值 |
| `message` | `string` | 消息内容 |

### 配置项列表
| 配置项ID | 类型 | 显示名称 | 说明 | 默认值 |
|----------|------|----------|------|--------|
| `title` | `input` | 组件标题 | 组件顶部显示的标题文本 | `'React 组件示例'` |
| `buttonText` | `input` | 按钮文本 | 主按钮显示的文本内容 | `'点击我'` |
| `primaryColor` | `colorPicker` | 主色调 | 组件的主题颜色 | `'#1890ff'` |
| `initialCount` | `inputNumber` | 初始计数值 | 计数器的初始值 | `0` |
| `message` | `input` | 消息内容 | 组件显示的消息文本 | `'这是一个 React 组件示例'` |

### 数据项列表
| 数据项名称 | 描述 | 数据结构 |
|-----------|------|----------|
| `data1` | 基础数据列表 | `Array<{ name: string, value: any }>` |

**data1 字段说明：**
- `name` (string)：名称
- `value` (any)：值
